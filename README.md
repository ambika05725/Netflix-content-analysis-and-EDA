# 🎬 Netflix Data Analysis Dashboard

## Project Description

This project is an interactive Netflix Dashboard developed using **Python, Pandas, Plotly, and Streamlit**. It allows users to filter Netflix content by **Country, Genre, and Release Year** and displays interactive charts.

## Features

- Filter by Country
- Filter by Genre
- Filter by Release Year
- Rating Distribution Chart
- Movies vs TV Shows Chart
- Interactive Dashboard


## How to Run

Run the following command:

```bash
streamlit run app.py
```

## Files

- `app.py` – Main Streamlit application
- `netflix_titles.csv` – Dataset
- `requirements.txt` – Required libraries
